# Food Delivery App (Android)
A fully functional **Food Delivery Android App** inspired by modern platforms like Zomato and Swiggy.
This app allows users to browse restaurants, order food, track orders, and make payments seamlessly.

---

## ✨Features
* Implemented user authentication, real-time restaurant listing, and seamless food ordering system.
* Integrated Firebase Realtime Database and Firebase Storage for efficient data management.
* Designed a modern and responsive UI using RecyclerView, ViewPager, and Material Design components.
* Built advanced features including banner slider, category-based filtering, cart system, and order tracking.
* Integrated push notifications (FCM) for real-time order updates.
* Implemented a simulated payment workflow to demonstrate end-to-end order processing.
* Developed a complete order management system for handling user orders.

---

## 🛠️ Tech Stack

* **Language**: Kotlin
* **Architecture**: MVVM
* **UI**: XML + RecyclerView + ViewPager
* **Backend**: Firebase Realtime Database
* **Authentication**: Firebase Auth
* **Storage**: Firebase Storage
* **Notifications**: Firebase Cloud Messaging (FCM)
* **Image Loading**: Glide
